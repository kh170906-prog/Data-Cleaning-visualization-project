# Data Cleaning & Visualization Project

## Overview
This project demonstrates basic data cleaning and visualization techniques using Python, Pandas, and Matplotlib in Google Colab.

## Dataset
The dataset contains student information such as:

- Student ID
- Age
- Study Hours
- Attendance
- Marks

## Objectives
- Load and inspect the dataset
- Identify missing values
- Remove duplicate records
- Perform statistical analysis
- Generate visualizations
- Analyze relationships between variables

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib

## Data Cleaning Steps
1. Loaded the dataset using Pandas
2. Checked for missing values
3. Removed duplicate records
4. Filled missing numerical values using the mean

## Visualizations
- Histogram
- Scatter Plot
- Correlation Matrix

## Key Findings
- Students with higher study hours generally scored better marks.
- Attendance showed a positive relationship with marks.
- The dataset was cleaned successfully and prepared for analysis.
